# 初始化网络梗知识库

这个脚本用于预填充网络梗和太鼓达人相关的知识到数据库中。

## 包含的知识

### 网络梗
- **董卓/吕布**: 来源于黄色同性恋漫画的网络梗
- **114514**: 恶臭先辈梗
- **1919810**: 网络数字梗
- **仲夏夜淫梦**: 网络文化梗
- **抽象/典/绷/乐**: 常用网络用语
- **搞抽象**: 2024-2025年流行的网络热梗
- **City不City**: 2024年夏天流行的网络热词
- **上春山**: 2024年央视春晚走红的歌曲
- **哪位少爷吸了**: 2025年11月引发的网络热梗
- **绷不住了/典中典**: 网络用语的完整表达

### 抽象文化和恶臭梗（低俗用语）
- **nmsl**: "你妈死了"的拼音缩写，抽象文化和恶臭梗的代表性用语
- **孙笑川**: 抽象文化的代表人物，网络主播
- **狗粉丝**: 孙笑川的粉丝群体，抽象文化的重要组成部分
- **抽象话**: 抽象文化的语言体系，特点包括符号化表达、谐音与方言
- **我佛了**: "我服了"的谐音表达，抽象话的典型特征
- **爷笑了**: 用"爷"自称，表示觉得好笑，抽象话的典型表达
- **带哥**: "大哥"的谐音，抽象话的常用称呼
- **🐎**: 抽象话中用来代表"妈"的表情符号
- **抽象工作室**: 抽象文化的起源，由孙笑川及其团队创建

### 太鼓达人知识
- **幽玄之乱**: 著名的魔王曲（10星难度）
- **第六条魔王**: 著名的魔王曲（10星难度）
- **紫煌ノ乱**: 由世阿弥（Zekken）创作的魔王曲
- **天狗囃子**: 由翡翠（Hisui）创作的歌曲
- **世阿弥/Zekken**: 著名曲师，创作了多首经典魔王曲
- **翡翠/Hisui**: 著名曲师，创作了多首经典歌曲
- **魔王曲**: 最高难度等级（10星难度）
- **曲师**: 歌曲创作者/作曲家

## 使用方法

### 方法1: 使用 Poetry（推荐）

```bash
poetry run python scripts/init_meme_knowledge.py
```

### 方法2: 直接使用 Python

确保已激活虚拟环境，然后运行：

```bash
python scripts/init_meme_knowledge.py
```

## 注意事项

1. **数据库连接**: 确保 MongoDB 正在运行，并且 `.env` 文件中的 `MONGODB_URL` 配置正确
2. **依赖**: 确保已安装所有项目依赖（`poetry install`）
3. **幂等性**: 脚本可以多次运行，已存在的条目会被更新而不是重复创建
